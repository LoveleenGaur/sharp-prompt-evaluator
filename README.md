# Prompt Engineering Lab

Prompt Engineering Lab is a learning and practice platform for prompt engineering.

The platform includes:
- Learn Prompt Engineering
- Evaluate Prompt
- Practice Lab
- Prompt Templates
- Progress Dashboard

Evaluation is powered by the SHARP Framework developed by Dr. Loveleen Gaur.

## Deployment
Deploy on Streamlit Community Cloud using:
- main file: `app.py`
